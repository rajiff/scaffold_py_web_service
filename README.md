
## Project initialization
```shell
uv init --name py_webservice --app --no-package  --python 3.12 .
```

followed by adding the following dependencies: FastAPI, Uvicorn (for running the server), and Pydantic for data validation in Python scripts using classes defined with pydantic models to define your API schema.  

Then you can run it

## Run
```shell
fastapi dev 
```
